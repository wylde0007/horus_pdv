Open source para operação de balcão

Frente de caixa e gestão comercial em um fluxo único.

Hórus PDV conecta vendas, estoque, caixa, clientes, fornecedores, usuários e relatórios em uma experiência web rápida para pequenos e médios comércios.

Explorar screenshots Conhecer recursos
React Frontend com Vite, TypeScript e Tailwind CSS
.NET 8 API ASP.NET Core com JWT e SQL Server
E2E Smoke test com Playwright para fluxo completo
MVP Base funcional para evolução fiscal e pagamentos

Operação completa sem espalhar o trabalho em várias ferramentas.

O sistema foi estruturado para reduzir fricção no balcão: venda rápida, conferência de caixa, cadastro assistido e gestão dos dados que alimentam o PDV.

Venda com baixa de estoque

Produtos vêm da API, entram no carrinho e baixam estoque ao confirmar a venda.

Caixa com regra operacional

A venda só inicia com caixa aberto e sessões vencidas são tratadas no fluxo.

Cadastros conectados

Clientes, fornecedores, produtos, empresa e usuários persistem no SQL Server.

Relatórios e histórico

Histórico de vendas com valores, recibo e reimpressão da última venda.

Segurança aplicada

Login com JWT, recuperação de senha, reCAPTCHA opcional e limite de tentativas.

Experiência de produto

Temas light/dark, tour guiado, componentes reutilizáveis e layout responsivo.

Módulos preparados para a rotina de loja.

Suba o ambiente local com SQL Server, API e frontend.

O projeto usa SQL Server em Docker, API ASP.NET Core e frontend React com Vite.

1. Banco

Inicie o SQL Server local.

docker start sqlserver2025

2. API

Execute a API .NET.

cd API/NETCORE
dotnet run

3. Frontend

Abra a interface web.

cd FRONTEND
npm install
npm run dev
